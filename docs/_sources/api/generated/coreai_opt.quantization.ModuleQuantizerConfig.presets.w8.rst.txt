w8
==

.. py:method:: w8(*, axis=None)
   :noindex:

   int8 weight-only quantization, per-channel symmetric.

   :param axis: Channel axis for per-channel quantization.
                When ``None`` (default), the axis is auto-resolved based on the module type
                during quantization.
   :type axis: int | None

   :returns: int8 weight-only module configuration.
   :rtype: ModuleQuantizerConfig
