w6
==

.. py:method:: w6(*, axis=0, group_size=16)
   :noindex:

   6-bit palettization, per-grouped-channel, group_size defaults to 16.

   :param axis: Channel axis to group along. Defaults to 0.
   :type axis: int
   :param group_size: Number of channels per palette group.
   :type group_size: int

   :returns: 6-bit palettization module configuration.
   :rtype: ModuleKMeansPalettizerConfig
